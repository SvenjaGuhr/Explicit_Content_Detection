# Explicit Content Detection in US-American Romance Novels
Developing a prediction model for the detection of explicit content, such as Sex and Nudity, and Violence and Scariness following the taxonomy of Tsai et al. 2024. 

Repository to accepted paper at NLP4DH:

Guhr, S. (2026). "Between Whispers and Screams: Loudness Standard Deviation as a Proxy for Explicit Content Detection in US Romance Novels." Proceedings of the 6th International Conference on Natural Language Processing for the Digital Humanities (NLP4DH 2026), ACL, San Diego/CA. **Preprint**: [Guhr_2026_NLP4DH_Paper_v.2.pdf](https://github.com/SvenjaGuhr/Explicit_Content_Detection/blob/main/Guhr_2026_NLP4DH_Paper_v.2.pdf) (26.06.2026).
