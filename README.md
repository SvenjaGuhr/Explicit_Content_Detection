# Explicit Content Detection in US-American Romance Novels
Developing a prediction model for the detection of explicit content, such as Sex and Nudity, and Violence and Scariness following the taxonomy of Tsai et al. 2024. 

Repository to accepted paper at NLP4DH: "Between Whispers and Screams: Loudness Standard Deviation as a Proxy for Explicit Content Detection in US Romance Novels"

The code and data is released under the GNU GPL v3.0 license. The author does not permit the use of the code or accompanying data in support of censorship, surveillance, or book-banning initiatives.
